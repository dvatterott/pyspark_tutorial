Tutorial Preparation.

  1. Install Docker Community Edition: https://docs.docker.com/install/
    • Those with older machines should install Docker Toolbox instead:
      https://docs.docker.com/toolbox/overview/

  2. Open your Terminal or Docker Quick Start Terminal.

  3. Run: docker run --rm -p 8888:8888 -p 4040:4040 dvatterott/pyspark_iris:intial_commit
    • On Windows - Note your ip-address at the top of the Terminal.

  4. The docker image will download (this takes ~20 minutes).

  5. A link will appear in your terminal. Enter this link into your browser.
    • On Windows, replace localhost with your ip-address.
